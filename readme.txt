This project contains NXT Brick code example for TechAwareness concerning LEGO Mindstorms NXT.
Programs are based on The LEGO®MINDSTORMS®NXT 2.0 Discovery Book by Laurens Valk

* leJOS subdirectory contains code written in Java using leJOS libraries and firmware
* NXT subdirectory contains LEGO MINDSTORMS NXT Snatcher program and blocks used within.
